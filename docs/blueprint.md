# **App Name**: QR Enrollment

## Core Features:

- Admin Form: Form for admins to input user details: name, email, and class.
- QR Code Generation: Generates a QR code instantly upon submission of the user details.
- Data Storage: Securely stores all registered user data.
- Searchable Table: Allows admins to view and manage user registrations through a searchable table.

## Style Guidelines:

- Primary color: Cornflower blue (#6495ED) to evoke professionalism and trust.
- Background color: Off-white (#F5F5F5) for a clean and neutral backdrop.
- Accent color: Steel blue (#4682B4) to provide contrast for interactive elements and highlights.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable appearance.
- Use simple and professional icons related to user management and QR codes.
- Maintain a clean and organized layout with clear visual hierarchy.
- Subtle animations or transitions for interactive elements to enhance user experience.
